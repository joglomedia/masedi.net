Outputs your blogroll to a page or post.

== Description ==

You can use the Blogroll to Page WordPress Plugin to output your blogroll links to any page or post on your blog by using its `blogroll-to-page` custom short code. The output options can be configured using parameters explained in the <a href='http://codex.wordpress.org/Function_Reference/wp_list_bookmarks#Parameters'>WordPress Codex</a> and set in the options page which is found under the `Links` menu. You can see an example of the output at our <a href='http://www.club-wp.com/wordpress-resources/'>WordPress Resources</a> page.

== Installation ==

1. Upload `blogroll-to-page.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Configure your output in `Links->Blogroll to Page` menu.
4. Place `[blogroll-to-page]` short code in your pages or posts.

== Frequently Asked Questions ==

= I have written post/page text before the short code, but the links are output before my text. =

Make sure you un-check the `Echo` option in admin page.
